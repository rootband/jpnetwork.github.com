명령어 히스토리 깨끗이 지우는 방법

본 글은 http://osxdaily.com/2012/12/17/how-to-clear-command-line-history/ 이 출처임을 밝혀두는 바입니다.

본딩(Bonding)이란?

여러개의 NIC을 이용하여 하나의 NIC으로 묶어서 사용하는 기술이며, 유용히 사용되어지고 있는 기술입니다. 본딩 방법은 추후 따로 작성하여 올릴 예정이나, 우선 모드에 대하여 어떤 것이 있는지 확인해보시기 바랍니다.

이는 모드에 따라서 여러 유형으로 변경될 수 있습니다.

Welcome to haroopress

Tuesday, February 21 2012 9:00 AM in Javascript

하루프레스를 이용한 블로그

Root.SO에 오신것을 환영합니다. 이 페이지는 하루프레스로 제작되어진 페이지이며, http://github.com/jpnetwork/jpnetwork.github.com과 연동되어지는 사이트입니다.

HarooPress to CentOS?

하루프레스는 Mac과 Ubuntu뿐만 아니라 CentOS 5.x이상 (REDHAT계열)에서 사용이 가능합니다. CentOS 5.x 버젼에서는 파이썬의 버젼이 낮으므로, 따로 Python을 업그레이드 하셔야 사용이 가능하십니다.

기본적으로 설치 하셔야 하는 것은 다음과 같습니다.

  • node 0.8.16
  • node-gyp 0.8.1
  • python 2.6.x
  • npm 최신버젼
  1. Python-2.5.6.tgz Source 설치
  2. curl http://npmjs.org/install.sh | sh
  3. git clone http://github.com/ry/node.git
  4. n 0.8.16
  5. git clone https://github.com/rhiokim/haroopress.git
  6. make init 에러발생 (robotskirt부분이 에러나면 정상)
  7. haroopress/Makefile내에 14번째줄 cd ./nodemodules/robotskirt;node-gyp rebuild를 아래와 같이 수정 cd ./nodemodules/robotskirt;node-gyp build
  8. vi node_modules/robotskirt/build/Makefile 수정 148번째줄 crsT를 crs로 수정
  9. haroopress 디렉토리에서 make init 수행
  10. 설정 세팅하시면 됩니다.
