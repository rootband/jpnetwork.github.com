ubuntu IP Setting

Ubuntu IP 설정하기

Ubuntu의 경우 Redhat과 달리 /etc/networking/interface에서 파일 하나로 네트워크가 설정이 가능합니다.

원격에서는 접속이 두절 될 수 있으므로, 확실하지 않을 경우 절때로 하지 마시기 바랍니다!!! 

$ sudo ifdown eth0 (일단 운영중인 상태의 데몬을 중지)
$ sudo vi /etc/network/interfaces
# This file describes the network interfaces available on your system
# and how to activate them. For more information, see interfaces(5).

# The loopback network interface
auto lo
iface lo inet loopback

# The Secondary network interface
auto eth0
iface eth0 inet static
address 192.168.11.100
netmask 255.255.255.0
#gateway 192.168.11.1 # eth1이 기본 Route가 되어야 하므로 주석처리
dns-nameservers 168.126.63.1 168.126.63.2

# The primary network interface
auto eth1
iface eth1 inet dhcp

# Static Route
up route add -net 192.168.11.0/24 gw 192.168.11.1 dev eth0

필자의 경우 2번째 NIC카드에 외부망을 연결하고 1번째 NIC에 내부망을 연동하였습니다.

따라서, 아이피가 필요하므로 # Static Route 밑 부분에 route를 추가해 준 것입니다.

Ubuntu에서는 따로 route를 해주는 경우가 없으므로 참조하시기 바랍니다.

모든 설정이 완료되었다면 아래와 같이 명령을 내려 재시작을 합니다.

sudo /etc/init.d/networking restart
