CentOS에 하루프레스 설치하기

하루프레스는 @rhiokim님께서 만드신 node기반의 정적 페이지 기반의 블로깅 엔진입니다.

하루프레스는 기본적으로 아래의 사항들을 이용하여 설치가 가능합니다.

CentOS Linux Version

5.x : python-2.4.3-46.el5 (Python 컴파일후 아래의 방법을 수행하시면 됩니다.)

6.x : python-2.6.6-29.el6 (사용가능)

github.com 계정 생성

하루 프레스 엔진은 노드와 6가지의 오픈소스의 조합으로 만들어졌습니다. 그렇다 보니 코어 엔진을 제외한 대부분의 모듈들이 Git 서브 모듈로 관리되고 있습니다.

물론 하루 프레스 도 Git 을 통해 관리되고 있습니다. 아래 공식 사이트에서 각 운영체제에 맞는 버젼을 설치하세요.

  • http://git-scm.com/

node 설치

cd /usr/local/src/
wget http://nodejs.org/dist/v0.8.2/node-v0.8.16.tar.gz
tar xvfz node-v0.8.16.tar.gz 
cd node-v0.8.16
./configure --prefix=/usr
make    
make install

Permission denied (publickey).

Permission denied (publickey)의 이유는 github.com에 상단 우측에 account setting에서 ssh-keys를 등록하지

않아 발생되는 것입니다. 사이트에서 방금 생성하셨던 /home/help/.ssh/id_rsa.pub 파일을 열어 해당 키를 넣어주시기 바랍니다.
